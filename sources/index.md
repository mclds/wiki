# 光梦

> light-dream (光明梦想，光之梦境，简称光梦)
> 光梦创立已有十年之久，前身是一个小服（梦殇之殛）发展而来，后续团队命名为 light dream studio (LDS)
> 经历了十年的分分合合，因为学业繁忙等各种原因导致没落，如今腐竹重启光梦计划，卷土重来。
