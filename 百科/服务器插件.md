# 服务器插件

## 领域插件

::: info
服务器自研领域插件，有`领域保护`，`领域传送`，添加`领域成员`，`成员权限`等一系列功能。
通过`区块`操作领域，可以对四周拓展与移除，当拓展方向上与其他人的领地相邻，则无法创建领域。
通过 `双击F键` 打开领域菜单，每个人会有领域拓展上限，领域购买需要消耗金币，通过菜单可以对领域进行更多的快捷操作。
:::
